About Image Gallery Project

Overview

Welcome to the Image Gallery Project! This project aims to provide users with a visually appealing way to explore and filter images from various categories. Whether you're looking for nature scenes, cityscapes, archival photos, or images related to spirituality or current events, our gallery has something for everyone.

Technologies Used

  • HTML, CSS, and JavaScript: The project is built using these fundamental web technologies to create the structure, style, and functionality of the gallery.
  • Unsplash API: We utilize the Unsplash API to dynamically fetch high-quality images from their vast collection based on user-selected filters.
  • FontAwesome: FontAwesome icons are used for adding visual elements such as the submit button and social media links.

How It Works

Upon visiting the gallery page, users are greeted with a selection of filter buttons representing different image categories. They can either click on these buttons to view images from a specific category or use the search bar to search for images based on keywords. Additionally, users can submit their own photos to be considered for inclusion in the gallery.
